## Setup

1. Clone this repo

```bash
git clone git@gitlab.com:devech/es-backend.git
```

2. Change Directory to `es-backend` folder.

```bash
cd es-backend
```

3. Install necessary packages by using `yarn` or `npm`.

```bash
yarn
```

```bash
npm install
```

4. Make a copy of `.env` file from `.env.example` file.

```bash
cp .env.example .env
```

5. Generate Key

```bash
adonis key:generate
```

### Migrations

Run the following command to run startup migrations.

```js
adonis migration:run
```

### Seedind

Run the following command to run startup migrations.

```js
adonis seed
```
